# 🧠 Ingredient Recommendation System (C)

## 🚀 Overview
A graph-based system that analyzes relationships between food ingredients using an adjacency matrix.  
It recommends related ingredients and suggests substitutions based on connection strength.

## 🛠️ Tech Stack
- C
- Graphs (Adjacency Matrix)
- Recursion
- Linked Lists

## ✨ Features
- Find directly related ingredients
- Discover ingredients within k-distance using recursion
- Filter ingredients based on restrictions
- Suggest optimal ingredient substitutions

## 🧩 Key Concepts
- Graph traversal
- Recursive algorithms
- Duplicate handling using linked lists
- Weighted relationships between nodes
